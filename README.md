# Personal Brand Site

> Dark editorial portfolio — type-driven, single column, zero clutter.

Single-file React site. No build step, no bundler, no framework overhead.

## Design system

- **Canvas**: `#0A0A0A`
- **Accent**: `#6FFF00` neon green
- **Typography**: DM Serif Display × JetBrains Mono
- **Layout**: 680px single column, generous whitespace
- Custom neon cursor with `mix-blend-mode: difference`
- Grain overlay at 3.5% opacity
- Staggered fade+rise load animation

## Stack

React 18 (CDN) · Babel Standalone · Google Fonts · Pure CSS
